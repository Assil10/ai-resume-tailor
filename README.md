# AI Resume Tailor 🤖

An n8n automation that takes a LaTeX resume and job description,
then generates a tailored resume and cover letter using AI.

## Stack
- n8n Cloud
- Groq API (Llama 3.3)
- JavaScript Code nodes

## How to use
1. Import the workflow JSON into n8n
2. Add your Groq API key
3. Update the Resume Store node with your resume
4. Submit the form with a job URL or description
